README

This repository contains the code to run the epidemiological model scenarios for:
"Modelling the health and economic impacts of M72/AS01E vaccination and BCG-revaccination: estimates for South Africa"

Open the project and run the script GenEpiOutput_complex.R to run the model

The model is implemented in an XML framework. Models are specified as XML files and simulated using our tbvax framework

Contents

countries - model XML files and other input files
processing_files - definitions of scenarios to run
R - the core code of the tbvax framework 
scripts - other scripts used in running and processing outputs
readme_output.txt - text file documenting the output structure
